## 🔑 Core Concepts (Visual Reference)

### The 3-Stage Architecture

```
┌──────────────────────────────────────────────────────┐
│               ANY CLAIM / DECISION                    │
│         "We should ban fossil fuels"                  │
└───────────────────┬──────────────────────────────────┘
                    │
    ┌───────────────┴───────────────┐
    │                               │
    ▼                               ▼
┌─────────────────┐          ┌─────────────────┐
│ STAGE 1: CAESAR │          │  STAGE 3: GOD   │
│   "What IS"     │          │  "What OUGHT"   │
│                 │          │                 │
│ • CO₂: 420 ppm  │          │ • Environment   │
│ • Temp: +1.1°C  │          │   priority?     │
│ • Economic cost │          │ • Economic      │
│   $X trillion   │          │   freedom?      │
│ • Job impact    │          │ • Future vs.    │
│   Y million     │          │   present?      │
└────────┬────────┘          └────────┬────────┘
         │                            ▲
         │   ┌─────────────────┐      │
         └──→│ STAGE 2: VERIFY │──────┘
             │  "Is it TRUE?"  │
             │                 │
             │ • SIP analysis  │
             │   of data       │
             │ • EBP challenge │
             │   to claims     │
             │ • Multi-source  │
             │   verification  │
             └─────────────────┘
```

**Key Insight:** Everyone sees *what is factually true* vs. *what values we prioritize*

**Traditional approach:** Mix facts and values → endless debate  
**S.V.E. approach:** Separate, then reconnect → clarity

### Epistemological Boxing Protocol (EBP)

```
┌─────────────────────────────────────────────────┐
│        HUMAN vs. AI ADVERSARIAL DIALOGUE         │
├─────────────────────────────────────────────────┤
│                                                  │
│  Round 1: Human proposes thesis                  │
│           "X is true because Y"                  │
│           ↓                                      │
│  Round 2: AI challenges from strongest angle     │
│           "But what about Z?"                    │
│           ↓                                      │
│  Round 3: Human refines or concedes              │
│           "You're right. X' is more accurate."   │
│           ↓                                      │
│  Repeat N times...                               │
│           ↓                                      │
│  Outcome: Vectorial purification                 │
│           Iterative Fact = survived N rounds     │
│                                                  │
│  Rules:                                          │
│  • Truth > Winning                               │
│  • Steel-man opponent                            │
│  • Virtuous concession                           │
│  • Falsifiable claims only                       │
└─────────────────────────────────────────────────┘
```

**Use case:** Before publishing paper, EBP with AI  
**Result:** Catch errors before peer review  
**Time saved:** Weeks of back-and-forth

### Socratic Investigative Process (SIP)

```
┌─────────────────────────────────────────────────┐
│          ITERATIVE TRUTH REFINEMENT              │
├─────────────────────────────────────────────────┤
│                                                  │
│  Initial Claim: "Vaccines cause autism"          │
│        ↓                                         │
│  Challenge 1: "What's the evidence?"             │
│        ↓                                         │
│  Refined: "Wakefield study showed correlation"   │
│        ↓                                         │
│  Challenge 2: "Was it replicated?"               │
│        ↓                                         │
│  Refined: "No, study was retracted"              │
│        ↓                                         │
│  Challenge 3: "What do meta-analyses show?"      │
│        ↓                                         │
│  Refined: "No causal link in 10+ studies"        │
│        ↓                                         │
│  Iterative Fact: "No evidence of causation"      │
│                                                  │
│  For complex topics → Meta-SIP                   │
│  (Multiple AI agents + synthesis)                │
└─────────────────────────────────────────────────┘
```

**Use case:** Fact-checking controversial claims  
**Result:** Transparent audit trail  
**Trust increase:** +40-60%
