## 🧭 The 3-Stage Architecture (Core Pattern)

```
┌─────────────────────────────────────────────────────────┐
│                    ANY CLAIM / DECISION                  │
└──────────────────────┬──────────────────────────────────┘
                       │
        ┌──────────────┴──────────────┐
        │                             │
        ▼                             ▼
┌──────────────────┐          ┌──────────────────┐
│ STAGE 1: CAESAR  │          │  STAGE 3: GOD    │
│  "What IS"       │          │  "What OUGHT"    │
│                  │          │                  │
│ • Empirical facts│          │ • Value judgment │
│ • Observable     │          │ • Ethics         │
│ • Measurable     │          │ • Vision         │
└────────┬─────────┘          └────────┬─────────┘
         │                             ▲
         │    ┌──────────────────┐     │
         └───→│ STAGE 2: VERIFY  │─────┘
              │  "Is it TRUE?"   │
              │                  │
              │ • SIP analysis   │
              │ • EBP challenge  │
              │ • Multi-agent    │
              └──────────────────┘
```

**Key Innovation:** Facts and values are **separated** but **connected** through transparent verification.

**Example:**
- **Claim:** "We should ban fossil fuels"
- **Caesar:** [CO₂ levels, temperature data, economic impact]
- **Verify:** [SIP checks data quality, EBP challenges interpretations]
- **God:** [Value weighing: environment vs. economy vs. freedom]

**Result:** Everyone sees *what is factually true* vs. *what we should do about it*.
